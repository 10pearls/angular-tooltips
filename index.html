Angularjs Tooltips

A simple pure angularjs directive
that generates a tooltip on your element
Put the mouse over the buttons

Usage

Html

Add the tooltips attribute to any of your elements
Hover me

Javascript

Add the tooltips module dependency
angular.module('app', [ '720kb.tooltips' ]);

Options

Tooltip position

You can set your tooltip to show on left or right or top or bottom position using the tooltip-side="" attribute
Top Bottom Left Right

Tooltip title

You can set your tooltip title (text/html doesn't matter) using the tooltip-title="" attribute or simply via title="" html attribute
Tooltip me

Tooltip content

You can set your tooltip content (text/html doesn't matter) using the tooltip-content="" attribute
Hover me

Tooltip size

You can set your tooltip size ( small or medium or large ) using the tooltip-size="" attribute
Small Medium Large

Tooltip speed

You can set the tooltip transition speed ( fast or medium or slow or int()milliseconds) ) using the tooltip-speed="" attribute
Tooltip fast Tooltip medium Tooltip slow Tooltip custom

Tooltip try

If space is not available for tooltip , it will automatically search for a similar alternative position to show. You can set tooltip try (1 || 0) using the tooltip-try="" attribute
Tooltip me Tooltip me

Tooltip lazy

If you don't want to re-init the tooltip position everytime the tooltip trigger events are fired, you can set tooltip lazy mode (true || false) using the tooltip-lazy="" attribute
//I will re-init my position everytime the mouseover event is fired Not Lazy //I will init my position on mouseover only the first time event is fired Lazy

Tooltip triggers

You can set your tooltip to show/hide on specific event/events, you can use the tooltip-show-trigger="" and the tooltip-hide-trigger="" attribute for this scope
Show tooltip only on click Hide tooltip only on click Show tooltip on click and mouseover and hide tooltip only on click

Tooltip CSS class

You can set a custom CSS class or a set of, using the tooltip-class="" attribute:
I will show a tooltip with class="tooltip-custom tooltip-for-me"
